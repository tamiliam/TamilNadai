## CAT1: WORD JOINING RULES

### 1. Single-letter noun + noun: JOIN with vallinam doubling
When a single-syllable noun (மா, பூ, தீ, etc.) is followed by a word starting with க/ச/த/ப, join and double the consonant.
- மா + கோலம் → மாக்கோலம்
- தீ + சுடர் → தீச்சுடர்
- தை + திங்கள் → தைத்திங்கள்
- தீ + பந்தம் → தீப்பந்தம்
- Exception words: நீ, கா, சா, சே, லீ, டை

### 2. Noun (without case marker) + postposition: JOIN
கீழ், குறித்த, குறித்து, பற்றி, பற்றிய, மீதான, மீது, மூலம், மேல், பின், பேரில், போல, போல், வசம் — join directly to the preceding noun.
- இது + பற்றி → இதுபற்றி
- கிளை + மீது → கிளைமீது

### 3. Noun WITH case marker + postposition: SEPARATE
When the noun has a case marker (இன், ஐ, etc.), separate the postposition.
- ஆட்சியின் + கீழ் → ஆட்சியின் கீழ் (not ஆட்சியின்கீழ்)
- நாளைக் + குறித்த → நாளைக் குறித்த (not நாளைக்குறித்த)
- இவற்றைப் + பற்றி → இவற்றைப் பற்றி (not இவற்றைப்பற்றி)
- அழைப்பின் + பேரில் → அழைப்பின் பேரில்
- இதைப் + போல → இதைப் போல (not இதைப்போல)

### 4. Noun + தோறும்: JOIN (with ம் → ந் change)
- வீடு + தோறும் → வீடுதோறும்
- வாரம் + தோறும் → வாரந்தோறும் (ம் becomes ந்)

### 5. Verb + auxiliary verb: JOIN (with vallinam doubling where needed)
Main verb (வினையெச்சம்) + auxiliary verbs (கட்டு, காட்டு, கொடு, கொள், தள்ளு, etc.) must be joined.
- பார்த்து + கொண்டிருந்தான் → பார்த்துக்கொண்டிருந்தான்
- வெட்டி + தள்ளு → வெட்டித்தள்ளு

### 6. Infinitive + causative verb: JOIN
- வர + செய் → வரச்செய்
- வர + வை → வரவை

## CAT2: INDIVIDUAL WORD CONVENTIONS

### 7. Verbal noun (-ல்) + ஆகாது ('should not'): JOIN
- கவலைப்படல் + ஆகாது → கவலைப்படலாகாது

### 8. Pronouns/nouns + இடையில்/இடையே ('between/among'): JOIN
- நம் + இடையில் → நம்மிடையில்
- உங்கள் + இடையே → உங்களிடையே
- மக்கள் + இடையே → மக்களிடையே
- இதற்கு + இடையில் → இதற்கிடையில்

### 9. Demonstratives அவ்/இவ்/எவ் + noun: JOIN
When followed by a vowel-initial word, the demonstrative joins with sandhi changes:
- அவ் + அமைப்பு → அவ்வமைப்பு (அ→வ)
- அவ் + ஊர் → அவ்வூர் (ஊ→வூ)
- அவ் + ஐயர் → அவ்வையர்
When followed by consonant-initial: join directly.
- அக் + காலம் → அக்காலம்

### 10. Verbal noun (செய்வது) + இல்லை: JOIN
- பார்ப்பது + இல்லை → பார்ப்பதில்லை

### 11. Infinitive (செய்ய) + இல்லை: JOIN with வி
- வர + இல்லை → வரவில்லை

### 12. Noun + இல்லாமல்: SEPARATE
- மழை + இல்லாமல் → மழை இல்லாமல் (not மழையில்லாமல்)
- Exception: மட்டும் + இல்லாமல் → மட்டுமில்லாமல் (JOIN)

### 13. அங்கு/இல்/இடம் + இருந்து ('from' meaning): JOIN
- அங்கு + இருந்து → அங்கிருந்து
- காலையில் + இருந்து → காலையிலிருந்து
- அவர்களிடம் + இருந்து → அவர்களிடமிருந்து

### 14. Word + இன்றி: JOIN
- உணவு + இன்றி → உணவின்றி
- அனுமதி + இன்றி → அனுமதியின்றி

### 15. Participial adjective (செய்த) + உடன்: JOIN
- திரும்பிய + உடன் → திரும்பியவுடன்

### 16. Verbal noun (செய்வது) + உண்டு: JOIN
- வருவது + உண்டு → வருவதுண்டு
- BUT noun + உண்டு: SEPARATE (மரணம் உண்டு, not மரணமுண்டு)

### 17. Noun + கு case + உரிய: JOIN
- பெண்களுக்கு + உரிய → பெண்களுக்குரிய

### 18. Noun + உள்ள: JOIN (possessive meaning)
- புறா + உள்ள → புறாவுள்ள
- கை + உள்ள → கையுள்ள
- வீடு + உள்ள → வீடுள்ள
- மனம் + உள்ள → மனமுள்ள
- BUT location meaning: SEPARATE (உலகில் உள்ள, not உலகிலுள்ள)

### 19. Noun + இன் + ஊடே/ஊடாக: JOIN
- கம்பிகளின் + ஊடே → கம்பிகளினூடே

### 20. Dative case + என்று ('for someone'): JOIN
- உனக்கு + என்று → உனக்கென்று
- என்ன + என்று → என்னவென்று

### 21. Word + என (particle meaning 'like/as'): JOIN
- புலி + எனப் → புலியெனப்

### 22. Dative case + ஏற்ற: SEPARATE
- உழைப்புக்கு + ஏற்ற → உழைப்புக்கு ஏற்ற (not உழைப்புக்கேற்ற)

### 23. Dative/verbal form + ஏன்: JOIN
- நமக்கு + ஏன் → நமக்கேன்
- வருவது + ஏன் → வருவதேன்

### 24. அப்படி/இப்படி/எப்படி + ஒரு: JOIN
- அப்படி + ஒரு → அப்படியொரு

### 25. Infinitive + ஒழிய: JOIN
- வந்தால் + ஒழிய → வந்தாலொழிய

### 26. Word + காட்டிலும் (comparison): SEPARATE
- நம்மைக் + காட்டிலும் → நம்மைக் காட்டிலும்

### 27. Word + கூடாது: SEPARATE
- போகக் + கூடாது → போகக் கூடாது (not போகக்கூடாது)

### 28. Infinitive + கூடிய: JOIN
- வரக் + கூடிய → வரக்கூடிய

### 29. Infinitive + கூடும்: JOIN
- இருக்கக் + கூடும் → இருக்கக்கூடும்

### 30. Infinitive + தக்க/தகாத/தகுந்த: SEPARATE
- போற்றத் + தக்க → போற்றத் தக்க (not போற்றத்தக்க)
- பேசத் + தகாத → பேசத் தகாத
- படிக்கத் + தகுந்த → படிக்கத் தகுந்த

### 31. Word + பார்க்கிலும்: SEPARATE
- வெயிலைப் + பார்க்கிலும் → வெயிலைப் பார்க்கிலும்

### 32. Participial adj + பிறகு: JOIN
- போன + பிறகு → போனபிறகு
- BUT with case marker: SEPARATE (அதற்குப் பிறகு, not அதற்குப்பிறகு)

### 33. Participial adj + பின்/பின்னர்/பின்பு/பின்னே/பின்னால்: JOIN
- பேசிய + பின் → பேசியபின்
- BUT with case marker: SEPARATE (கதவுக்குப் பின், யானையின் பின்)

### 34. Participial adj + புதிதில்: SEPARATE
- வந்த + புதிதில் → வந்த புதிதில் (not வந்தபுதிதில்)

### 35. Noun + பூர்வமாக/பூர்வமான: JOIN
- உணர்வு + பூர்வமான → உணர்வுபூர்வமான

### 36. Participial adj + போதிலும்: JOIN
- அடிபட்ட + போதிலும் → அடிபட்டபோதிலும்

### 37. Participial adj/noun + போது/பொழுது: JOIN
- வரும் + போது → வரும்போது
- விழுந்த + போது → விழுந்தபோது
- தேர்தலின் + போது → தேர்தலின்போது

### 38. Word + போலும் (sentence-end): SEPARATE
- வீடு + போலும். → வீடு போலும்.

### 39. Word + போன்ற/போன்று: SEPARATE
- அவரைப் + போன்ற → அவரைப் போன்ற

### 40. Participial adj + மட்டில்/மட்டும்: JOIN
- பொறுத்த + மட்டில் → பொறுத்தமட்டில்

### 41. Noun + மயம்/மயமாக: JOIN
- உணர்வு + மயமாக → உணர்வுமயமாக

### 42. Infinitive + மாட்டு: JOIN
- வர + மாட்டேன் → வரமாட்டேன்

### 43. Participial adj + மாத்திரத்தில்: SEPARATE
- பார்த்த + மாத்திரத்தில் → பார்த்த மாத்திரத்தில்

### 44. Participial adj + முகமாக: JOIN
- வந்த + முகமாக → வந்தமுகமாக

### 45. Word + முடியும்/முடியாது: SEPARATE
- வர + முடியும் → வர முடியும் (not வரமுடியும்)
- தடுக்க + முடியாது → தடுக்க முடியாது

### 46. Case marker + முன்/முன்பு/முன்பே/முன்னே: SEPARATE
- தொலைபேசியின் + முன் → தொலைபேசியின் முன்
- நுழைவதற்கு + முன் → நுழைவதற்கு முன்

### 47. Participial adj + வண்ணம்: JOIN
- வந்த + வண்ணம் → வந்தவண்ணம்

### 48. All word types + வரை/வரைக்கும்/வரையில்: JOIN
- மாலை + வரை → மாலைவரை

### 49. இடது/வலது + word: SEPARATE
- இடது + புறம் → இடது புறம் (not இடதுபுறம்)

### 50. Word + வழியாக: SEPARATE
- கடல் + வழியாக → கடல் வழியாக (not கடல்வழியாக)

### 51. Word + வாக்கில்: JOIN
- நீள + வாக்கில் → நீளவாக்கில்

### 52. Accusative case + விட (comparison): JOIN
- ஆற்றலை + விட → ஆற்றலைவிட

### 53. வெகு + word: JOIN
- வெகு + காலம் → வெகுகாலம்

### 54. Infinitive + வேண்டும்/வேண்டாம்: SEPARATE
- வர + வேண்டும் → வர வேண்டும் (not வரவேண்டும்)
- வர + வேண்டாம் → வர வேண்டாம்

### 55. வேறு + ஒரு/ஒருவர்: JOIN with sandhi
- வேறு + ஒரு → வேறொரு
- வேறு + சில → வேறுசில
- வேறு + இடம் → வேறிடம்

## CAT3: SANDHI (VALLINAM DOUBLING)

The four hard consonants (வல்லெழுத்து): **க, ச, த, ப**

When certain words/forms are followed by a word starting with க/ச/த/ப, the hard consonant must be doubled (ஒற்று மிகும்) by inserting க்/ச்/த்/ப் before the following word.

### Must double (ஒற்று மிகும்):

**After அந்த/இந்த/எந்த:**
- அந்த + காலம் → அந்தக் காலம்
- இந்த + தேர்தல் → இந்தத் தேர்தலில்
- எந்த + பக்கம் → எந்தப் பக்கம்

**After செய்ய-type infinitive (ending in அ):**
- படிக்க + கொடுத்தார் → படிக்கக் கொடுத்தார்
- செய்ய + சொன்னார் → செய்யச் சொன்னார்

**After adverbs (சரிவர, கூட, மிக, போல, என, ஏற்ப):**
- மிக + சரியான → மிகச் சரியான
- கூட + பயந்து → கூடப் பயந்து

**After -ஆக suffix:**
- அழகாக + பேசி → அழகாகப் பேசி
- காரமாக + சமையல் → காரமாகச் சமையல்

**After சின்ன:**
- சின்ன + குழந்தை → சின்னக் குழந்தை
- சின்ன + பையன் → சின்னப் பையன்

**After negative participial adjective (ஈறுகெட்ட எதிர்மறை பெயரெச்சம்):**
- நிறைவேறா + கனவுகள் → நிறைவேறாக் கனவுகள்
- தீரா + பகை → தீராப் பகை

**After பற்றி:**
- பற்றி + கேள் → பற்றிக் கேள்
- பற்றி + பேசினார் → பற்றிப் பேசினார்

**After மாதிரி + noun:**
- மாதிரி + காகிதம் → மாதிரிக் காகிதம்

**After செய்து-type verbal participle (ending in இ/உ):**
- விட்டு + கொடு → விட்டுக் கொடு
- ஓடி + போ → ஓடிப் போ
- கேட்டு + சொல் → கேட்டுச் சொல்

**After compound nouns (உ-ending noun + noun):**
- மருந்து + கடை → மருந்துக் கடை
- சொத்து + சண்டை → சொத்துச் சண்டை

**After அங்கு/இங்கு/எங்கு (optional but preferred):**
- அங்கு + கண்டாயா → அங்குக் கண்டாயா

**After 2nd case marker ஐ (accusative):**
- பழத்தை + கடித்து → பழத்தைக் கடித்து
- கடமையை + செய்ய → கடமையைச் செய்ய
- கதவை + தட்டி → கதவைத் தட்டி
- கையை + பிடித்து → கையைப் பிடித்து

**After 4th case marker கு (dative):**
- குழந்தைக்கு + காய்ச்சல் → குழந்தைக்குக் காய்ச்சல்
- வீட்டுக்கு + செல்ல → வீட்டுக்குச் செல்ல

## Quick Reference: JOIN vs SEPARATE

| Context | Rule | Example |
|---------|------|---------|
| Noun (no case) + postposition | JOIN | இதுபற்றி |
| Noun WITH case + postposition | SEPARATE | இவற்றைப் பற்றி |
